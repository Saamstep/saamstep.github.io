Import a Minecraft Resource Pack

  • Windows
  • Mac

1. Go to %appdata%\.minecraft
2. Open the resourcepacks folder
3. Drag and drop the folder (in ZIPPED form) into the resourcepacks folder.
4. Open Minecraft (in 1.9 or above)
5. Go to Options > Resource Packs
6. Choose Rainbow!

1. Open Finder 
2. Click Go to Folder and type in ~/Library/Application Support/minecraft. 
3. Open the resourcepacks folder 
4. Drag and drop the folder (in ZIPPED form) into the resourcepacks folder. 
5. Open Minecraft
6. Go to Options > Resource Packs. 
7. Choose Rainbow!
